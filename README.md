# IntellMeet

## AI-Powered Enterprise Meeting and Collaboration Platform

### Project Description
IntellMeet is an AI-powered enterprise meeting and collaboration platform that enables users to schedule meetings, conduct video conferences, communicate in real time, and generate AI-powered meeting summaries and action items.

### Features
- Secure Login and Signup
- Video Meeting
- Meeting Scheduling
- AI Meeting Summary
- Dashboard and Analytics
- Notifications
- Real-time Chat
- User Authentication

### Technology Stack
- React.js
- TypeScript
- Vite
- Node.js
- Express.js
- MongoDB
- Socket.io
